# 画像のトリミングとリサイズ

ブラウザ上で画像を読み込み、トリミングとリサイズを行うシンプルなWebアプリです。
静的サイトとして動作しますが、ブラウザの制限を避けるため `file://` で直接開くよりサーバー経由で使うことを推奨します。
ローカルで使う場合は、Node.js と付属のパッケージ管理ツール npm を使って同梱の簡易サーバーを起動できます。

## 機能

- ファイル選択、ドラッグ&ドロップ、クリップボード貼り付けで画像を読み込み
- トリミング比率は `自由` `オリジナル` `正方形` `16:9`
- マウスドラッグでクロップ範囲の作成、移動、サイズ変更。矢印キー（位置）とshift+矢印キー（サイズ）で微調整可能
- リサイズは縦横比固定で`幅(px)`、`高さ(px)`、`%` 指定
- クリップボードへのコピー、PNGダウンロード、バッチ処理用のコマンドを出力可能
- 新規画像の読み込み時に、可能なら前回のトリミング位置・サイズなどの設定を引き継ぎます

## 使い方

```bash
npm start
```

ブラウザで `http://localhost:4173` を開いて使います。

## 一括処理

同じトリミング・リサイズ設定を複数画像に適用する CLI も用意しています。「一括処理コマンドをコピー」でクリップボードにコマンドが入ります。

```bash
npm install
npm run batch -- --crop 100,80,1200,800 --resize width --width 640 image1.jpg image2.png
```

ディレクトリを指定すると、中の画像ファイルをまとめて処理します。出力先はデフォルトで `batch-output` です。

```bash
npm run batch -- --crop 0,0,1080,1080 --resize none --out-dir out ./images
```

主なオプション:

- `--crop x,y,width,height`: トリミング範囲
- `--resize none|width|height|percent`: リサイズ方法
- `--width 640`: 幅指定リサイズ
- `--height 480`: 高さ指定リサイズ
- `--percent 50`: パーセント指定リサイズ
- `--format png|jpeg|webp`: 出力形式
- `--out-dir out`: 出力先ディレクトリ

## ファイル構成

- `index.html` UI
- `styles.css` スタイル
