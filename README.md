# 画像のトリミングとリサイズ

ブラウザ上で画像を読み込み、トリミングとリサイズを行うシンプルな Web アプリです。
静的サイトとして動作しますが、ブラウザの制限を避けるため `file://` で直接開くよりサーバー経由で使うことを推奨します。
ローカルで使う場合は、Node.js と付属のパッケージ管理ツール npm を使って同梱の簡易サーバーを起動できます。

## 機能

- ファイル選択、ドラッグ&ドロップ、クリップボード貼り付けで画像を読み込み
- トリミング比率は `自由` `オリジナル` `正方形` `16:9`
- マウスドラッグでクロップ範囲の作成、移動、サイズ変更
- リサイズは縦横比固定で `幅(px)` `高さ(px)` `%` 指定
- 出力プレビューと PNG ダウンロード

## 使い方

```bash
npm start
```

ブラウザで `http://localhost:4173` を開いて使います。

## ファイル構成

- `index.html` UI
- `styles.css` スタイル
- `app.js` 画像編集ロジック
- `server.js` 静的ファイル配信用の最小サーバー
